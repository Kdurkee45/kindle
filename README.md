# Kindle

AI-powered application factory. Describe what you want in a sentence, answer a
few questions, and get a complete, runnable project.

## How It Works

```
Grill → Research → Architect → Dev → QA → Package
```

1. **Grill** — The machine asks you focused questions with recommended answers.
   Your spec emerges from the conversation, not from a blank page.
2. **Research** — Investigates the technology landscape, libraries, patterns.
3. **Architect** — Designs the system and splits work into parallel dev tasks.
4. **Dev** — Multiple agents build simultaneously, each in their own directory.
   Tests are written alongside code.
5. **QA** — Technical checks (lint, types, tests, format) + product audit
   (anti-slop detection). Self-healing loop back to Dev if issues found.
6. **Package** — Ensures everything runs, generates README, delivers a
   complete project.
